                                      WORDS MONSTER

______________________________________________________________________________________

Words Monster is a hangman-style word guessing game developed in Python using Pygame.

The player must guess letters to find a hidden word. Each incorrect letter contributes to the creation of a monster. The goal is to find the word before the monster is fully created.
Gameplay

    A word is randomly selected
    The player guesses letters using the keyboard
    Correct letters are revealed in the word
    Incorrect letters reduce the number of lives
    The game ends when the word is found or when all lives are lost

Difficulty Levels

    Easy: short words, 7 lives
    Medium: long words, 7 lives
    Hard: long words, 5 lives
    God Like: long words, random number of lives

Technologies

    Python
    Pygame

Executable Version

The game can be compiled into a Windows executable (.exe) so it can be played without installing Python.
